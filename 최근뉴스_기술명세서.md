# 최근뉴스 기술 명세서

기존 알람 웹앱(간편 알람)에 "최신 뉴스 검색 + 읽어주기" 기능을 추가하기 위한 기술 검토 문서입니다.

## 1. 요구사항

- 최신 뉴스 5개를 검색하여 화면에 표시
- 표시된 뉴스를 음성으로 읽어주는 기능 제공
- 기존 알람 웹앱(React/HTML 아티팩트) 내에 통합

## 2. 뉴스 검색 — 네이버 뉴스 검색 API

최신 뉴스 5개 검색은 **네이버 뉴스 검색 API(Naver Search API - News)** 를 사용합니다.

- 네이버 개발자센터에서 애플리케이션 등록 후 Client ID / Client Secret 발급 필요
- 엔드포인트: `https://openapi.naver.com/v1/search/news.xml` 또는 `.json`
- 요청 시 `X-Naver-Client-Id`, `X-Naver-Client-Secret` 헤더 필요
- 브라우저(클라이언트)에서 직접 호출 시 CORS로 막히고, Client Secret이 노출되므로 **반드시 서버(프록시)를 경유**해야 함 → 아래 3번 항목 참고

### 프록시 서버 — Supabase

네이버 API 호출 시 CORS 문제 해결 및 Client Secret 보호를 위해 **Supabase Edge Functions**를 프록시 서버로 사용합니다.

- React 앱 → Supabase Edge Function → 네이버 뉴스 검색 API 순으로 요청 전달
- 네이버 Client ID/Secret은 Supabase 환경변수(Secrets)에 저장하여 클라이언트에 노출되지 않도록 처리
- Edge Function에서 응답을 받아 필요한 필드(제목, 요약, 링크, 출처, 발행일 등)만 정제하여 React로 반환
- 필요 시 Supabase Database를 함께 사용해 검색 결과 캐싱/로그 저장도 고려 가능

### 프론트엔드 — React

- 뉴스 카드 UI, 검색 결과 상태 관리(useState/useEffect 등)는 React로 구현
- Supabase Edge Function을 `fetch`로 호출하여 뉴스 데이터를 받아온 뒤 리스트/카드 형태로 렌더링

## 3. 뉴스 읽어주기 — Web Speech API (SpeechSynthesis)

브라우저 내장 기능으로 API 키나 외부 요청이 필요 없으며, `speechSynthesis.speak()` 한 줄로 구현 가능합니다.

- 장점: 무료, 즉시 사용 가능
- 주의점: 한국어 음성 품질/지원 여부가 사용자의 OS·브라우저에 따라 다를 수 있음 (예: 크롬 데스크톱은 대체로 양호, 일부 모바일 브라우저는 음성이 어색할 수 있음)

## 4. 대안 (더 자연스러운 음성이 필요한 경우)

TTS 품질을 높이고 싶다면 외부 TTS 서비스(네이버 클로바더빙, ElevenLabs 등)를 연동하는 방법도 있습니다. 다만 이 경우 API 키 발급 및 클라이언트에서 직접 호출이 필요해 구조가 더 복잡해집니다. 처음에는 Web Speech API로 시작하고, 추후 필요 시 업그레이드하는 것을 권장합니다.

## 5. 알람 연동 — 자동 뉴스 읽기

알람 시간이 되면 최신 뉴스 읽기가 이어지도록 연동합니다.

### 고려사항

1. **브라우저 자동재생 정책**: `speechSynthesis.speak()`는 대부분의 브라우저에서 사용자 상호작용(클릭 등) 없이는 자동 재생이 제한될 수 있음 → 완전 자동 실행보다는 사용자 액션을 트리거로 활용
2. **탭/백그라운드 제약**: 웹앱 특성상 브라우저 탭이 열려 있어야 알람 및 후속 동작이 정상 동작함 (네이티브 앱과 달리 완전한 백그라운드 실행은 어려움)
3. **네트워크 타이밍**: 뉴스 API 호출 → 응답 → TTS 재생까지 약 1~2초 지연 발생 가능 → 로딩 UI 처리 필요

### 추천 흐름

1. 알람 울림 (알람 소리 재생)
2. 사용자가 알람을 끄는 클릭 이벤트를 트리거로 뉴스 API 호출 시작
3. 응답 수신 후 카드 UI 표시 + 첫 번째 뉴스부터 자동으로 `speechSynthesis`로 읽기 시작
4. "다음 뉴스", "정지" 등 컨트롤 버튼 제공

이 방식은 완전한 자동 실행은 아니지만, 사용자 입장에서는 "알람을 끄자마자 뉴스가 읽힌다"는 경험을 제공하여 요구사항과 사실상 동일한 효과를 냅니다.

## 6. 기술 스택 요약

| 기능 | 추천 기술 |
|---|---|
| 뉴스 검색 5개 | 네이버 뉴스 검색 API |
| 프록시 서버 | Supabase (Edge Functions) |
| 프론트엔드 | React |
| 뉴스 읽어주기 | Web Speech API (`speechSynthesis`) |
